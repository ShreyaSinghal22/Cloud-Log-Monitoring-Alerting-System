# AWS Cloud Logging Project

## Project Overview

This project demonstrates the implementation of a serverless cloud logging solution using AWS services. The system captures events through an AWS Lambda function and generates logs that are monitored through Amazon CloudWatch.

The objective of this project is to understand how logging, monitoring, and serverless computing work together in a cloud environment to improve observability and troubleshooting.

---

## Problem Statement

Monitoring applications and cloud resources is essential for identifying errors, tracking system behavior, and ensuring reliable operations.

Traditional logging systems often require dedicated servers and manual management. This project utilizes AWS serverless services to automate log generation and monitoring without managing infrastructure.

---

## Objectives

- Learn AWS Lambda fundamentals
- Implement serverless event processing
- Generate application logs automatically
- Monitor logs using Amazon CloudWatch
- Understand cloud monitoring and observability concepts
- Gain hands-on experience with AWS services

---

## AWS Services Used

### AWS Lambda
Executes the serverless function responsible for generating and processing logs.

### Amazon CloudWatch
Collects, stores, and displays logs generated by the Lambda function.

### AWS IAM
Provides secure permissions and access control for Lambda execution.

---

## Project Workflow

1. An event triggers the Lambda function.
2. The Lambda function executes the business logic.
3. Log messages are generated during execution.
4. CloudWatch automatically captures these logs.
5. Logs can be monitored and analyzed through the AWS Console.

## Implementation Steps

### Step 1: Create IAM Role

Created an IAM role with permissions for:

- AWS Lambda Execution
- CloudWatch Logs Access

### Step 2: Create Lambda Function

Configured a Lambda function using Python.

### Step 3: Add Logging Statements

Implemented logging within the Lambda code to capture execution details and events.

### Step 4: Deploy Function

Uploaded and deployed the Lambda function to AWS.

### Step 5: Monitor Logs

Verified log generation through Amazon CloudWatch.
